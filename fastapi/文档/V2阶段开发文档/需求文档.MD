# 多智能体全栈开发平台产品需求文档

## 1. 项目概述和目标

### 1.1 项目背景
设计并实现一个以多智能体协同为核心的AI全栈开发平台，面向初学者和学生，通过模拟软件工程中的真实角色分工，辅助用户高效完成全栈项目开发。

### 1.2 项目目标
- 提供高度自由的智能体选择机制
- 降低全栈开发的入门门槛
- 为后续功能扩展预留空间
- 在现有用户系统基础上构建平台框架

### 1.3 目标平台
Web应用（浏览器访问）

## 2. 用户画像和使用场景

### 2.1 目标用户
初学者和学生群体

### 2.2 主要使用场景
- 用户选择合适的智能体进行项目开发
- 自由切换不同智能体获取专业建议
- 通过智能体学习全栈开发流程

## 3. 功能需求清单

### 3.1 主页改造
**功能描述：** 在现有用户系统基础上，将主页改造为导航型页面
**具体要求：**
- 顶部导航栏：首页 + AI开发平台 + 用户中心
- 主页中央大卡片展示"AI开发平台"功能
- 卡片内容：标题 + 简短描述 + "进入平台"按钮
- 采用GitHub风格配色方案
- 为后续功能扩展预留布局空间

### 3.2 智能体展示页面
**功能描述：** 展示三个核心智能体，用户可自由选择
**具体要求：**
- 三个智能体卡片并排展示：
  - 产品经理智能体
  - 后端开发智能体  
  - 前端开发智能体
- 每个卡片包含：角色名称 + 一句话描述 + "开始对话"按钮
- 右上角"使用说明"按钮
- 点击智能体卡片新标签页打开对应对话链接

### 3.3 使用说明功能
**功能描述：** 为用户提供平台使用指导
**具体要求：**
- 右上角明显的"使用说明"按钮
- 点击后右侧滑出说明面板
- 面板内容：各智能体作用 + 建议使用顺序（硬编码）
- 点击其他区域或关闭按钮收起面板
- 为后续文档系统预留扩展接口

### 3.4 智能体链接集成
**功能描述：** 将现有智能体对话系统集成到平台
**具体要求：**

## 设计风格规范 - 现代极简风格

### 视觉特征
- **配色方案**：纯白背景 + 深灰色文字(#333) + 浅灰色辅助色(#666) + 蓝色强调色(#007AFF)
- **布局特点**：大量留白，宽松的间距，呼吸感强
- **卡片设计**：简洁的白色卡片，轻微圆角(8px)，细微阴影
- **按钮风格**：圆角按钮，蓝色填充，白色文字，hover时轻微变深
- **字体排版**：无衬线字体，层次分明的字号体系

### 具体应用
- **主页大卡片**：居中布局，充足的内边距，简洁的描述文字
- **智能体卡片**：等宽排列，统一的卡片高度，清晰的层次
- **导航栏**：简洁的顶部导航，适中的高度，清晰的分割
- **使用说明面板**：右侧滑出，白色背景，优雅的动画效果
