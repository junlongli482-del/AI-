## 📋 接口模块总览

### v1 用户系统 (12个接口)

| 模块 | 路径前缀 | 接口数 | 主要功能 |
|------|----------|--------|----------|
| 用户注册 | `/api/v1/user_register` | 3 | 注册、用户名邮箱验证 |
| 用户认证 | `/api/v1/user_auth` | 3 | 登录、登出、Token刷新 |
| 用户中心 | `/api/v1/user_profile` | 4 | 个人信息、昵称管理 |
| 密码管理 | `/api/v1/password_manager` | 2 | 密码修改、强度检测 |

| 模块 | 路径前缀 | 接口数 | 主要功能 |

| 文档管理 | `/api/v2/document_manager` | 8 | 文档CRUD、文件夹管理 |
| 文件上传 | `/api/v2/file_upload` | 8 | 文件上传、验证、历史 |
| MD编辑器 | `/api/v2/md_editor` | 11 | 在线编辑、AI优化 |
